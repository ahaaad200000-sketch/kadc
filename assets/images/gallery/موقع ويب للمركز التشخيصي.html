📱 5. تحسين التصميم للجوال (Responsive)

أضف هذا داخل <head> في كل صفحة:

html
<meta name="viewport" content="width=device-width, initial-scale=1.0">

ثم أضف إلى style.css:

css
@media (max-width: 768px) {
    body {
        padding: 10px;
    }

    header h1, header h2 {
        font-size: 22px;
    }

    section {
        padding: 10px;
    }

    input, textarea {
        width: 100%;
        box-sizing: border-box;
    }
}

    let map, infoWindow;

    function initMap() {

        center: centerPosition,
        zoom: 15  // مستوى التكبير

        if (navigator.geolocation) {

3. ملاحظات مهمة

  • يجب أن تحصل على مفتاح API من Google Maps وتفعّل خدمة Maps JavaScript API. بدون المفتاح، لن تعمل الخريطة مع الدبوس أو تحديد الموقع.

  • استبدل YOUR_API_KEY بمفتاحك الحقيقي.

  • تأكد من أن صفحة موقعك تعمل عبر https لأن navigator.geolocation غالبًا لا يعمل على صفحات غير آمنة.

  • الإحداثيات (centerLat, centerLng) يجب أن تكون دقيقة جدًا لموقع المركز — إذا كان لديك العنوان، يمكنك استخدام Google Maps لتحديد النقطة، اضغط يمين → "ما هذا المكان" (What's here) لتحصل على إحداثيات دقيقة.

  • زر “أظهر موقعي” سيطلب من المستخدم الإذن لتحديد موقعه — إذا رفض، ستظهر رسالة خطأ.

إذا تحب، يمكنني مساعدتك في:

  • توليد مفتاح Google Maps أو شرح كيفية الحصول عليه.

  • تحديد الإحداثيات الدقيقة إذا أرسلت لي العنوان التفصيلي.

  • تحسين تصميم زر “أظهر موقعي” أو تخطيطه.
